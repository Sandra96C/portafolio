## ⚙️ Instalación

Clona el repositorio:

```bash
git clone https://github.com/Sandra96C/portfolio.git
```

Entra en la carpeta:

```bash
cd portfolio
```

Instala las dependencias:

```bash
npm install
```

Ejecuta el proyecto:

```bash
npm run dev
```

## 📦 Build

```bash
npm run build
```
