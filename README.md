## 📖 Sobre el proyecto

Este portfolio reúne mi experiencia como desarrolladora, los proyectos en los que he trabajado y las tecnologías que domino. Está construido desde cero con React y Vite, con un diseño responsive (mobile-first), animaciones en CSS y una arquitectura de componentes reutilizables mediante CSS Modules.

**Características principales:**

- 📱 Diseño totalmente responsive (móvil, tablet y desktop)
- 🎨 Animaciones de entrada y micro-interacciones en scroll
- 🧩 Componentes reutilizables y desacoplados por sección
- ♿ Atención a la accesibilidad (roles ARIA, `prefers-reduced-motion`, navegación por teclado)

## ⚙️ Instalación

Clona el repositorio:

```bash
git clone https://github.com/Sandra96C/portafolio.git
```

Entra en la carpeta del proyecto:

```bash
cd portafolio
```

Instala las dependencias:

```bash
npm install
```

Ejecuta el proyecto en modo desarrollo:

```bash
npm run dev
```
